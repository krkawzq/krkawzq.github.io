# 个人主页

一个现代化的个人页面框架，包含响应式设计和流畅的用户体验。

## 功能特性

- ✨ 现代化的UI设计
- 📱 完全响应式布局，支持移动端和桌面端
- 🎨 流畅的动画效果
- 🚀 平滑滚动导航
- 💫 滚动触发的动画效果

## 文件结构

```
.
├── index.html      # 主页面文件
├── styles.css      # 样式文件
├── script.js       # JavaScript交互文件
└── README.md       # 说明文档
```

## 使用说明

1. 打开 `index.html` 文件，根据你的个人信息修改内容：
   - 更新导航栏中的品牌名称
   - 修改首页的个人介绍和名字
   - 更新"关于我"部分的描述
   - 修改技能卡片中的技能列表
   - 更新项目部分，添加你的实际项目
   - 修改联系部分的链接和邮箱

2. 自定义样式：
   - 编辑 `styles.css` 文件中的 CSS 变量来更改主题颜色
   - 修改 `:root` 中的颜色变量来匹配你的品牌色

3. 添加图片：
   - 可以创建 `assets` 文件夹存放图片
   - 替换头像占位符和项目图片占位符

## 部署

### GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择主分支作为源

### 其他平台

可以直接将文件上传到任何静态网站托管服务。
